Mapping::__construct (   $sLocal = null,
array  $aParams = array() 
)
final

Create a new mapping instance. You can specify the following types of local-mapping: absolute: (starting with a /) A mapping relative to the root. relative: (not starting with a /) A mapping relative to the directory of the current mapping. current: (current mapping) Specify null (default-value).

Parameters
string$sLocalThe local mapping.
