Detailed Description

Helper class to contain and manage a list of specific items. Originally created for UserManager, but ported to Watena main.

It's purpose is to have a simple and consistent way to keep a collection of objects without replicating similar code. Main features:

  • Associativeness based on user-specifiable key
  • Callback for verifying add operations
  • Auto sorting when items are added
  • Iterator implementation
  • Easy access to first, last or default item(s)

Container::addItem ( IContainerItem  $oItem)

Try to add the given item to the container. Checks if no items with an equal key exists, and (if set) checks the verify-add callback. Call the addedToContainer method on $oItem.

If a sorting function is set, auto sorts the items after adding, but before the callback.
