PHP exception catcher. This function will try to redirect the exception to its rightfull owner/logger, and will try to create a log entry for the provided data. Logger::setFilter ( ILogFilter  $oFilter)
final

Set an optional filter (object implementing ILOgFilter) to verify and adapt log messages and calls. It to provides the means to change the data before being logged, thus enabling external log-control.

Only one filter for each logger can be active at any given time. If during the filtering, the filter would be changed, the behaviour is undefined.
