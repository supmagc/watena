Detailed Description

Holds all filterdata required to match a filter to a mapping, and knows which model, view, controller to load.

Files are discovered by using FilterGroup, and are parsed with the help of FilterData and FilterRule.

A filter file should be located within a 'filters' directory within a library, and have a name beginning with filter. and ending on .xml
