DbConnection::buildWhere (   $mId,
  $mIdField,
  $sConcatenation = 'AND' 
)

This method builds the select/where part of a query in a format that supports PDO query-variables. The building is semi smart for binary comparisons ans 'null statements.

You can specify a single value for $mId and $mIdField, or you can specify array values (none associative arrays) with equal count.

If the value in Id is null the comparison will become 'IS' or 'IS NOT'

You can specify the following binary directives for IdField: !: Becomes <> in the query <: Becomes < in the query >: Becomes > in the query =: Becomes = in the query

DbConnection::connect ( )

Make the actual connection. If a connection with an equal identifier, dsn, user and password allready exists, this will return an existing PDO instance. If no open connection is found, This will initialize a new PDO instance:

  • Not a persistent connection, as this failed on subsequent requests
  • Error mode switched to exception
  • Connection will take the system encoding from Encoding::charset()
  • Connection sill take the system timezone
  • Connection will have a timeout of 120
